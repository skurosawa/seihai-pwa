# Seihai

入力 → 整理 → 行動  
思考を3ステップで整えるシンプルな思考整理アプリ。

Seihai は、頭の中にある考えやタスクをそのまま書き出し、  
並べ替え、削り、最後に「次の一手」を明確にするためのツールです。

---

## Demo

https://skurosawa.github.io/seihai-pwa/

---

## Features

- 3ステップ構成（入力 / 整理 / 行動）
- 横スワイプによる画面切り替え
- ドラッグ＆ドロップで並び替え
- スワイプ削除 + Undo
- 自動保存（localStorage）
- 共有機能（Web Share API / クリップボード対応）
- PWA対応（ホーム画面に追加可能）

---

## Use Cases

- 頭の中のモヤモヤを整理したいとき
- アイデアを具体的な行動に落とし込みたいとき
- タスクの優先順位を見直したいとき
- 思考をシンプルに言語化したいとき
