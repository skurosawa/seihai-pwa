# Seihai

「入力 → 整理 → 行動」で思考をまとめる、iPhone向けPWA。

🔗 Demo  
https://skurosawa.github.io/seihai-pwa/

---

## 使い方

### 1. 入力
- textarea に入力
- Enterで追加（改行は分割されます）
- Shift+Enterで改行

### 2. 整理
- ドラッグで並び替え
- 左スワイプで削除

### 3. 行動
- 整理結果から行動を生成
- 共有ボタンでMarkdown共有

---

## 特徴
- Safari最適化
- PWA対応（ホーム画面追加可）
- 自動保存（localStorage）
